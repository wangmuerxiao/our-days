# 朋友纪念日倒计时页面

一个为好友纪念日准备的简洁静态页面，包含双人头像、心跳动画、倒计时卡片以及背景音乐。页面的设计最初来自 QQ 好友，在此基础上做了多次优化和微调，便于二次创作或直接部署。

## 功能亮点

- **纪念日倒计时**：展示与纪念日相距的天数、小时、分钟等信息，采用卡片式布局。
- **心跳与液态玻璃视觉**：中央心形图标配合透明玻璃效果，营造纪念氛围。
- **双人头像区**：支持替换头像，框架和倒影效果已就绪。
- **背景与音乐**：固定背景图及自动播放的背景音乐，可根据需要替换。
- **移动端适配**：使用弹性布局，手机与桌面端都有良好显示效果。

## 目录结构

- `index.html`：入口页面，包含基础结构、样式引用与 DOM 布局。
- `css/styles.css`：主要样式文件，包括玻璃拟态、卡片与自适应布局的实现。
- `js/`：脚本资源，负责倒计时逻辑和页面动态效果。
- `image/`：示例背景及头像资源，可按需替换。

## 使用方式

1. 直接在浏览器打开 `index.html` 即可查看效果，无需额外构建步骤。
2. 如需部署到静态服务器（例如 GitHub Pages、Vercel 等），将整个目录上传即可。

## 自定义指南

- **纪念日时间**：在 `js` 文件中调整目标日期，以匹配你的纪念日。
- **头像与背景**：替换 `image/` 下的图片文件，或在 `index.html` 中调整图片路径。
- **样式与动画**：修改 `css/styles.css` 以更改配色、玻璃效果和心跳动画参数。
- **音乐**：在页面中替换音频文件或链接，保持自动播放逻辑即可。
