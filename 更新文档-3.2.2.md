## 本次修改概述

本次版本主要对页面的名字配置方式和倒计时说明文字的显示效果进行了优化，方便后续维护与版本检查。

## 修改内容

### 1. 名字显示改为分开配置

之前页面中的名字显示存在配置混用问题，不方便单独调整不同位置的名称。

本次已改为只保留两组名字配置：

- 头像下方名字
- 标题括号中的名字

当前 `data.json` 中使用以下字段：

```json
{
  "maleAvatarName": "他",
  "femaleAvatarName": "她",
  "maleBirthdayTitleName": "他",
  "femaleBirthdayTitleName": "她"
}
```

字段说明：

- `maleAvatarName`：男生头像下方显示的名字
- `femaleAvatarName`：女生头像下方显示的名字
- `maleBirthdayTitleName`：`你来这人间的一天（...）` 中括号里的名字
- `femaleBirthdayTitleName`：`光落在你身上的日子（...）` 中括号里的名字

说明：

- 已移除多余的旧名字字段，避免出现三组名字配置造成混乱。
- 后续如果要改名字，只需要修改 `data.json` 即可。

### 2. 倒计时说明文字颜色优化

为了解决部分背景下文字不清晰的问题，本次将以下说明文字统一调整为黑色：

- `还有xx天`
- `自那日起，时光悄悄溜走了 xx 次朝霞与暮霭`
- 周年进度百分比文字

本次仅调整文字颜色，没有修改布局与文案结构。
